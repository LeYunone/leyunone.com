# 医理自救指南

接受一个年轻灵魂来自，医院、病理、治疗...等的洗礼后的吐槽；

`说是总结更好听`

**2023-08-20**

## 支付

```markdown
医保当在卫生服务区、医院中使用有定向医保规则：
1、分小点、大点、中医 
2、每个选择对应的医院一年只能修改一次
3、非选择中的医院无法使用医保
```

## 治疗

### 打针

```
氯化钾\钾水类 点滴：
当他们超过300ML时就要注意时间了，因为 K 有渗透细胞内外差的效果，即点滴速度太快的话就会痛
```

```
点滴的水可以试试常用手，因为弱用手撑头或者拿手机都比较舒适
```

```
在资源不紧张的医院，在打针前可以根据点滴时间，和医生提出需要开一张床位的需求
```

### 腹泻

```
诺氟沙星真不能多吃，一定要警听医生指导；
并且我的副作用是会导致头产生比较严重的偏头痛，靠休息无法解决，不过可以靠布洛芬中和
```

```
恢复阶段时，会产生出很想吃酸的意识。
并且会过多的分泌口水
但是！不要吃酸的食物，建议买营养快线顶顶就可以了。
我吃了酸萝卜之后，当晚腹泻加重
```

## 预防

```
运动过度，即每天出大量汗但是补充不够
会产生炎症
首先是发烧预警，如果继续则发展为炎症导致的细菌感染肠胃炎
```
